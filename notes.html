Lab 2:2 HTML Knowledge

HTML tags are hidden keywords in a webpage. Examples are: <html>, <title>, and <body>.

An empty tag is a tag that is not closed with a slash character. A regular tag contains the opening tag, content, and closing tag.

An HTML tag attribute comes in name/value pairs and provides additional information about HTML elements.

To nest an HTML tag, you would have to close the tag that was opened recently.

In order to add the id attribute, you would have to choose an HTML element that is unique.

To add a class to an HTML tag, you would have to add <div> elements to each HTML class of your choice.

An ID is for one unique element, while the class attribute can be used for multiple elements.

A type selector selects all elements of the given type on a page.

The head tag contains information in the head portion of a document. Some examples are: <meta>, <title>, and <style>.

The body tag lists the document's body. Some examples are text, images, and links.

How to calculate the height and width:

"The total width of an element should be calculated like this:

Total element width = width + left padding + right padding + left border + right border + left margin + right margin.

The total height of an element should be calculated like this:

Total element height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin."
